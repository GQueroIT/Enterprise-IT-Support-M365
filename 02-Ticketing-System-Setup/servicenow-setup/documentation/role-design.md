# Role and Queue Design

## Support Groups Created

### Tier 1 Help Desk
Primary service desk queue for end-user incidents.

### Desktop Support
Endpoint troubleshooting and workstation support.

### Server Administration
Infrastructure and server-related escalation queue.

### Network Operations
Routing, switching, and connectivity issues.

### Security Operations
Security incident escalation queue.

## Escalation Model
Tier 1  
→ Desktop  
→ Server/Network  
→ Security
